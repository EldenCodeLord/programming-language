Programming Languages!

Trying to decide which programming language to start with?

Learning coding languages can be very overwhelming and stressful. There are so many languages to choose from. It helps to start off with the basics. By no means does "basic" mean easy. Every programming language has it's strengths and complexities.

Here is a list of programming languages!:

  • Ruby
  • C#
  • JavaScript (JS)
  • HTML
  • Python
  • CSS
  • Swift

Noticed how JavaScript, HTML, and CSS is in bold letters? That's because I believe that these are good programming languages to begin with. They will teach you basic terminology and syntax structures.
